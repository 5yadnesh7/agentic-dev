# Lessons learned (self-learner)

Append here when something repeats or when a step took longer than expected. The agent (or you) can read this before similar tasks and update skills if needed.

## Format

- **Date** | **What** | **Lesson** (what worked / what to do differently)
- Optional: which skill to update or threshold to change.

## Example

- 2025-03-05 | Auth PRD | Always clarify token expiry and refresh flow in Phase 1; saved time in LLD.
- 2025-03-05 | BE-02 implement | Took 45m; consider splitting "auth middleware" into two tasks (middleware + tests) in task board next time.
