# Idea: Todo SaaS

A simple todo-list SaaS for teams. Users sign up, create lists, share with teammates, and manage tasks.

## Core features

- User signup/login (email + password)
- Create, edit, delete todo lists
- Add tasks to lists; mark done
- Share lists with teammates (read/write)
- Mobile-responsive web UI

## Target users

- Small teams (2–20 people)
- Personal productivity

## Non-goals (v1)

- Real-time collaboration
- Integrations (Slack, etc.)
- Mobile apps (web-first)
