# Quick Start

## 1. Clone repo

```bash
git clone https://github.com/your-org/agentic-dev.git
cd agentic-dev
```

## 2. Open project in Cursor

Open the `agentic-dev` folder in Cursor.

## 3. Start a new project

Type in the chat:

```
Idea: Build a SaaS for event booking
```

## 4. Cursor executes the new-project workflow

The **dev-supervisor** orchestrates: research → assumption validation → roadmap → spec → architecture → task plan → execute.

## 5. Generated artifacts

Outputs appear in:
- `docs/` — research, roadmap, architecture, product spec
- `tasks/` — task files (001-X.md, 002-Y.md, …)
- `memory/project-state.md` — current phase, completed tasks, decisions, lessons

---

**Next:** Run `Build: [your idea]` for the full bootstrapping path, or use triggers like `Bug:`, `Review:`, `Spec:` — see [README](README.md#triggers).
